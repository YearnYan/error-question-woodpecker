# FEATURES 研究：错题啄木鸟

## Table Stakes（不做就不成立）

- 上传错题并自动抽取题干与图形区域。
- 错题分析输出：知识点、考点、答案解析。
- 生成 3 类题：相似题、变式题、综合应用题。
- 题目与图形语义匹配，图形清晰可打印。
- 作业纸式预览，支持导出/打印。
- 基础性能与稳定性保障（超时、重试、缓存、监控）。

## Differentiators（形成优势）

- 8 学科统一渲染协议与学科插件机制。
- 图文分离架构的自动路由（公式前端，图形后端）。
- 图形-题干一致性校验器（避免“题图错配”）。
- 出卷风格模板库（学校样式、年级样式、学科样式）。
- 生成质量打分（考点一致性、题型差异度、可解性）。

## Anti-Features（当前不做）

- 社交互动、排行榜、社区评论。
- 大而全教务系统集成。
- 浏览器端复杂图形编辑器。
- v1 阶段的超多模板个性化（先保证标准模板质量）。

## 复杂度与依赖

- 高复杂度：后端图形渲染编排、缓存与任务容错。
- 中复杂度：多学科提示词策略与质量校验。
- 中复杂度：学校作业纸版式系统与分页。
- 核心依赖：AI 模型服务、TeX/Python 渲染容器、对象存储/CDN。
