# 错题啄木鸟（错题作业生成网站）

## What This Is

这是一个面向 K-12 学生和教师的错题作业生成网站。用户在左侧上传错题（含可能的图形），系统先给出错题分析，再基于同一考点生成 3 类“举一反三”题目，并在右侧以真实学校作业纸样式预览。系统覆盖数学、物理、化学、生物、地理、英语、政治、历史 8 个学科。

## Context

- 目标场景：课后错题巩固、教辅训练、教师快速出卷。
- 学科覆盖：数学、物理、化学、生物、地理、英语、政治、历史。
- 关键输入：拍照上传题目（可能含几何图、坐标图、实验装置图、地图、时间轴等）。
- 关键输出：AI 分析 + 三类题作业单页（图文准确匹配）。
- 架构红线：
  1) 纯文本公式前端渲染（KaTeX/MathJax）；
  2) 复杂图形后端渲染（LaTeX/TikZ/Python/Puppeteer）并以 SVG 下发 CDN。

## Constraints

- **渲染策略**: 必须执行图文分离，保证性能、稳定性和视觉一致性。
- **学科准确性**: 8 学科题干与图形必须精准匹配，错配会直接损害可用性。
- **性能**: 预览区首屏与交互需快速可见，避免长时间白屏/阻塞。
- **稳定性**: 图形渲染需支持失败重试与降级，不能因单图失败阻断整份作业。
- **输出质量**: 作业纸版式需接近真实学校模板，包括题号、间距、分页、留白。

## Key Decisions

| Decision | Rationale | Outcome |
|----------|-----------|---------|
| 采用“前端公式 + 后端图形”双通道渲染 | 前端轻量快，后端图形可控且可打印 | ✅ Good |
| 统一输出 SVG 图形 | 可缩放、清晰、便于排版与 CDN 缓存 | ✅ Good |
| 三类题固定顺序输出 | 保证学习路径稳定：相似 -> 变式 -> 综合应用 | ✅ Good |
| 先做单题到单页作业闭环 | 快速验证核心价值，降低首版复杂度 | ✅ Good |
| 分析结果采用结构化字段 + 一致性检查 | 提升可测试性并降低题外前提风险 | ✅ Good |
| 8 学科统一渲染协议（学科插件化） | 避免烟囱式实现，便于扩科与维护 | ⏳ Pending |
| 渲染运行时指标内建 | 保障上线前可观测性与故障定位效率 | ✅ Good |
| 移动端左右栏切换 | 在窄屏下保障核心流程可达 | ✅ Good |
