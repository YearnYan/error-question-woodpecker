# 学科渲染调研：政治（道德与法治/思想政治，K-12）

## 1）学科渲染特征概述

政治学科题目以文本为主，但高频出现“关系结构图 + 数据材料图 + 案例流程图”。  
排版痛点：

- 材料题文字量大，且常伴随政策关系图、经济统计图。
- 逻辑链条（原因-措施-结果）若仅文本呈现，学习迁移效率低。
- 图文风格若偏“互联网海报”，会偏离学校试卷语境。

## 2）前端渲染范围（纯文本公式类）

### 2.1 必须且只能走前端渲染的符号/表达类型（按学段）

初中（道德与法治）：

- 法律概念关键词、条文编号、案例要点编号。
- 简单逻辑连接词标注（因为/所以、如果/那么）。

高中（思想政治）：

- 经济学基础表达式（如 `GDP=C+I+G+NX` 的文本表达）。
- 增长率、比重、占比等代数表达。
- 逻辑符号表示（`P \rightarrow Q`、`P \land Q`）用于论证训练。

统一规则：

- 纯文本、纯符号、纯代数表达全部前端渲染。
- 一旦进入“关系网络可视化”与“多主体交互图”，必须后端。

### 2.2 标准 LaTeX 示例

```latex
GDP = C + I + G + NX

r=\frac{X_t-X_0}{X_0}

P \rightarrow Q,\quad P \land Q
```

### 2.3 前端渲染性能评估与宏包建议

- KaTeX 足够支持政治学科中偶发的符号表达。
- 性能重点在长材料文本渲染与题块交互，不在公式复杂度。
- 建议：
  - 前端实现“材料段落锚点 + 题目引用定位”。
  - 对引文与结论使用统一样式块，提升可读性。

## 3）后端渲染范围（复杂图形类）

### 3.1 K-12 图形类型穷举

- 初中：
  - 事件流程图（规则形成、行为后果）
  - 角色关系图（家庭、学校、社会角色）
- 高中：
  - 宏观经济循环图（生产-分配-交换-消费）
  - 政策传导路径图（政策工具 -> 市场主体 -> 社会结果）
  - 组织结构图（国家机构职责关系）
  - 议题因果图（问题成因、对策、预期影响）
  - 统计材料图（柱状、折线、饼图）

### 3.2 图形 -> 后端技术栈映射

- 关系图/流程图/组织结构图：
  - `Mermaid` / `Graphviz`
- 经济循环示意图（含箭头和反馈环）：
  - `TikZ`（可控箭头与版式）
- 数据统计图：
  - `Matplotlib` / `ECharts SSR`

### 3.3 输出要求

- 图形应保持“教材风格”，避免过度视觉装饰。
- 因果方向必须明确，箭头不可歧义。
- 图中术语与题干术语完全一致，避免同义替换导致理解偏差。

## 4）边界与特殊情况处理（Edge Cases）

- 边界 1：题干仅给出 GDP 公式和数字  
  决断：前端文本公式即可。

- 边界 2：材料题包含“简短因果链（3 节点）”  
  决断：若题目要求结构化分析，建议后端流程图；否则可前端列表。

- 边界 3：简单对比表（A 政策 vs B 政策）  
  决断：前端表格可行；若包含动态因果关系，改后端图。

- 边界 4：政治题中嵌入地理/历史数据图  
  决断：图表仍按对应学科规则后端渲染。

- 边界 5：多主体博弈题  
  决断：主体 >2 且关系多向时，必须后端关系图。

## 附：政治学科落地建议

- 建立“论证结构模板库”：总分、并列、递进、因果、对策五类。
- 输出题目时附带“答题结构提示”（非标准答案），提升训练可迁移性。
- 材料题图形与题目小问建立锚点映射，方便学生定位证据。
