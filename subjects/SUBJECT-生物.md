# 学科渲染调研：生物（K-12）

## 1）学科渲染特征概述

生物题并非“公式重学科”，但高度依赖结构图、流程图、统计图与实验图。  
主要痛点：

- 图示信息密集（细胞结构、生态网络、遗传图解），图文错位会直接影响判题。
- 常出现“图+表+文字”多模态题型，普通前端排版稳定性不足。
- 高中生物强调过程与调控机制，流程关系图比公式更关键。

## 2）前端渲染范围（纯文本公式类）

### 2.1 必须且只能走前端渲染的公式/符号类型（按学段）

初中：

- 基础比值与统计表达（增长率、比例、频数等）。
- 光合作用/呼吸作用的文本反应式（不含结构图）。

高中：

- 遗传概率表达式（分离定律、自由组合的基础概率写法）。
- 种群增长模型的文本公式表达（课标范围内）。
- 实验数据计算表达（均值、误差基础表示）。

统一规则：

- 只要是代数符号、概率表达、文本反应式，全部前端渲染。
- 不涉及生物结构空间关系或流程图拓扑时，必须前端渲染。

### 2.2 标准 LaTeX 语法示例

```latex
% 遗传概率
P(aa)=\frac{1}{4},\quad P(A\_)=\frac{3}{4}

% 简单增长表达
N_t=N_0(1+r)^t

% 光合作用文本表达（示意）
\ce{6CO2 + 6H2O -> C6H12O6 + 6O2}
```

### 2.3 前端渲染性能评估与宏包建议

- KaTeX 足以覆盖生物中的绝大多数符号表达。
- 若出现化学式，启用 `mhchem`。
- 性能重点不在公式，而在图文混排：
  - 前端只承担轻量公式与文本。
  - 图形区域采用后端 SVG 占位异步填充。

## 3）后端渲染范围（复杂图形类）

### 3.1 K-12 图形类型穷举

- 初中：
  - 细胞结构示意图（植物细胞/动物细胞）
  - 人体系统结构图（消化、呼吸、循环等）
  - 食物链/食物网示意
- 高中：
  - 光合作用与呼吸作用过程图
  - 神经-体液-免疫调节示意图
  - 遗传图解、系谱图、染色体行为示意
  - 生态系统能量流动与物质循环图
  - 实验流程图、结果统计图（柱状/折线/散点）

### 3.2 图形 -> 后端技术栈映射

- 生物流程图/调控网络图：
  - `Mermaid`（后端渲染为 SVG）
  - `Graphviz`（复杂有向关系）
- 结构示意图（细胞、器官、系统）：
  - `TikZ`（模板化图元）
- 遗传系谱图：
  - `Graphviz` 或 `TikZ`（节点关系可控）
- 实验统计图：
  - `Matplotlib` / `pgfplots`

### 3.3 输出要求

- 图形需可读且“教材风格化”：标注清楚、色彩克制、避免装饰噪声。
- 生物部件命名与题干一致（术语不混用）。
- 对流程图强制有方向箭头，避免歧义。

## 4）边界与特殊情况处理（Edge Cases）

- 边界 1：Punnett 方格（孟德尔遗传方格）  
  决断：简易 2x2 可前端表格；涉及多基因或图文联动时统一后端。

- 边界 2：光合作用方程式 + 叶绿体结构图  
  决断：方程式前端，结构图后端，双通道并排。

- 边界 3：生态题仅给“箭头文字链”  
  决断：若考察关系推理，仍建议后端生成标准食物网图。

- 边界 4：实验题数据表  
  决断：纯表前端；需要趋势判断的图像后端。

- 边界 5：英文缩写与拉丁学名混排  
  决断：文本前端；若依赖拓扑关系展示，后端图中统一字体处理。

## 附：生物学科落地建议

- 建立“流程图语义模板库”（调节、循环、传递三类）。
- 对每类图配置“最小可读标准”：节点数上限、最小字号、箭头清晰度。
- 在生成链路中加入“概念一致性检查”（器官/组织/过程层级不混乱）。
