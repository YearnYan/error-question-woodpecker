# 学科渲染调研：地理（K-12）

## 1）学科渲染特征概述

地理题的核心载体是地图与统计图。  
排版痛点：

- 题干高度依赖底图（经纬网、等高线、行政区、区域定位）。
- 同题常叠加多图层（地形 + 气候 + 人口/产业数据）。
- 坐标轴、图例、比例尺、指北针缺失或偏差会直接导致误判。

## 2）前端渲染范围（纯文本公式类）

### 2.1 必须且只能走前端渲染的公式/符号类型（按学段）

初中：

- 比例尺换算公式（文字代数表达）。
- 经纬度表示法的符号文本（`30^\circ N` 等）。
- 基础统计计算表达（平均值、增长率）。

高中：

- 地理计算中的代数表达（人口增长、区位指数等课标范围内简式）。
- 时区换算、太阳高度角简化表达（文本公式层面）。

判定规则：

- 只要是纯计算表达，不依赖地图空间关系，必须前端渲染。
- 与图层叠加、区域定位、地形关系相关的内容一律不在前端绘图。

### 2.2 标准 LaTeX 语法示例

```latex
% 比例尺换算
\text{实地距离}=\text{图上距离}\times \text{比例尺分母}

% 增长率
r=\frac{N_t-N_0}{N_0}

% 经纬度文本
30^\circ N,\quad 120^\circ E
```

### 2.3 前端渲染性能评估与宏包建议

- KaTeX 可覆盖地理中的符号级表达，性能负担较小。
- 地理性能瓶颈通常不在公式，而在地图渲染和图像加载。
- 前端侧仅保留轻量文本与公式，地图图形全部走后端 SVG。

## 3）后端渲染范围（复杂图形类）

### 3.1 K-12 图形类型穷举

- 初中：
  - 等高线地形图、地形剖面图
  - 经纬网定位图、区域示意图
  - 气温曲线与降水柱状图（气候图）
- 高中：
  - 大气环流与洋流示意图
  - 工农业区位示意图、城市功能分区图
  - 人口金字塔、迁移流向图
  - 资源分布图、产业链空间布局图
  - 多图层叠加专题图（自然 + 人文）

### 3.2 图形 -> 后端技术栈映射

- 地图底图与投影：
  - `Cartopy`（投影、经纬网、底图要素）
- 地理专题图与统计图：
  - `Matplotlib`（柱状/折线/散点/组合图）
  - `ECharts SSR`（复杂图例后端导出 SVG）
- 流向与路径图（迁移、交通、洋流）：
  - `D3-geo` 或 `Matplotlib` + 自定义箭头层
- 地形剖面/等值线：
  - `Matplotlib contour` + SVG 输出

### 3.3 输出要求

- 必须明确比例尺、指北针、图例、坐标刻度。
- 地图色彩使用教育友好配色，避免误导性的视觉强调。
- 同题多图层需共享统一地理边界与坐标参考。

## 4）边界与特殊情况处理（Edge Cases）

- 边界 1：仅有“比例尺换算”文字题  
  决断：前端公式，不生成地图。

- 边界 2：题干给出简易方位关系（东南西北）  
  决断：若不要求读图，可前端文本；若要求空间定位，必须后端图。

- 边界 3：气候数据表  
  决断：纯表可前端；要求“判断气候类型”时应后端生成标准气候图。

- 边界 4：行政区轮廓极简示意  
  决断：只要涉及区域判断，依然走后端，保证轮廓与标注准确。

- 边界 5：在线瓦片地图调用  
  决断：题目渲染不依赖在线交互地图，统一后端静态 SVG，确保打印一致性。

## 附：地理学科落地建议

- 建立“地理图层规范”：底图层、数据层、标注层、辅助层分离。
- 区分课堂解释图与考试作答图，考试图优先信息密度与可读性。
- 对区域题加入“图例完整性检查”和“坐标系完整性检查”。
