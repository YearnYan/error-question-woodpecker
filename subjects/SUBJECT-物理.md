# 学科渲染调研：物理（K-12）

## 1）学科渲染特征概述

物理题的视觉核心是“模型图 + 公式 + 单位”的三位一体。  
K-12 典型排版痛点：

- 同一道题经常包含受力图/电路图/实验装置图与多行公式推导。
- 图中箭头方向、正负极、刻度、参考系一旦错误会导致答案完全变化。
- 题目中常见“图 + 表 + 文”联合描述，网页端极易出现对齐错乱。

## 2）前端渲染范围（纯文本公式类）

### 2.1 必须且只能走前端渲染的公式/符号类型（按学段）

初中（7-9 年级）：

- 速度、密度、压强、功和功率等基础公式文本表达。
- 欧姆定律、串并联等效关系的代数式。
- 热学基础公式、比热容表达式。

高中（10-12 年级）：

- 运动学、牛顿定律、动量守恒、机械能守恒公式。
- 电场/磁场/电磁感应相关符号表达。
- 光学成像公式、波动与振动公式。
- 原子物理基础表达（频率、能级差、德布罗意关系等课标范围内表达）。

统一判定：

- 纯符号方程、单位换算式、文字物理量定义均前端渲染。
- 不含空间结构关系、线路连接关系、受力方向关系时，必须前端渲染。

### 2.2 标准 LaTeX 语法示例

```latex
% 速度与加速度
v=\frac{\Delta s}{\Delta t},\quad a=\frac{\Delta v}{\Delta t}

% 牛顿第二定律
F=ma

% 欧姆定律
I=\frac{U}{R}

% 功与功率
W=Fs\cos\theta,\quad P=\frac{W}{t}

% 焦耳定律
Q=I^2Rt

% 薄透镜成像公式
\frac{1}{f}=\frac{1}{u}+\frac{1}{v}
```

### 2.3 前端渲染性能评估与宏包建议

- 默认：KaTeX（题干中高频短公式渲染成本低）。
- 兜底：MathJax（复杂环境或兼容性需求）。
- 建议宏包：
  - `siunitx` 风格单位书写规范（前端可通过宏近似处理）。
- 性能策略：
  - 公式与单位分层渲染，避免整段频繁重排。
  - 对同题中重复公式做缓存键（学科 + 公式字符串）。

## 3）后端渲染范围（复杂图形类）

### 3.1 K-12 图形类型穷举

- 力学：
  - 受力分析图（重力、支持力、摩擦力、拉力）
  - 斜面、滑轮组、杠杆、机械效率示意
  - 位移-时间、速度-时间、加速度-时间图像
- 电学：
  - 串并联电路图、伏安法测电阻实验图
  - 含电流表、电压表、滑动变阻器、开关、灯泡的综合电路
- 光学：
  - 凸透镜/凹透镜成像光路图
  - 平面镜、折射、全反射示意
- 热学与实验：
  - 温度-时间变化曲线
  - 热量实验装置图、状态变化示意
- 高中拓展图：
  - 带电粒子在电磁场中的轨迹示意
  - 振动与波形图、相位关系图

### 3.2 图形 -> 后端技术栈映射

- 电路图：
  - `circuitikz`（LaTeX 宏包，电学符号标准化）
- 受力图/几何化力学图：
  - `TikZ`
- 光路图：
  - `TikZ`（射线、法线、折射角标注）
- 实验装置与示意流程：
  - `TikZ` + `Graphviz`（当结构偏流程图化）
- 运动学图像（s-t/v-t/a-t）：
  - `pgfplots` 或 `Matplotlib`

### 3.3 输出要求

- 全部输出 SVG，线宽、箭头、字体统一模板化。
- 电路图必须保持器件符号规范，不允许自由手绘风格。
- 对需要数值读取的图（斜率/面积）必须保证坐标刻度准确。

## 4）边界与特殊情况处理（Edge Cases）

- 边界 1：仅出现 `F=ma`、`P=UI` 等短公式  
  决断：前端公式渲染，不走后端。

- 边界 2：题干含“文字描述电路连接关系”但未配图  
  决断：生成题若涉及连接判断，必须后端补全规范电路图。

- 边界 3：v-t 图像仅一条直线  
  决断：只要题目要求“读图/求斜率/求面积”，统一后端渲染。

- 边界 4：实验题带简单数据表  
  决断：纯表格可前端；若需图像化（拟合线、误差棒）走后端。

- 边界 5：多图联动（装置图 + 数据图）  
  决断：整题图形统一后端批处理，避免前后端坐标与样式不一致。

## 附：物理学科落地建议

- 在题目生成后增加“物理维度校验”：方向、单位、数量级、符号约定。
- 对电学和光学设立独立渲染模板库，先覆盖高频考型再扩展。
- 对图像题建立可读性规则：关键点标注、坐标网格、题干变量名一致。
