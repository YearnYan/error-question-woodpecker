# 数学-题目图形约束规范

## 1. 适用范围与目标

本规范适用于数学试题中的几何图、坐标系图、函数图像、统计图表、
数轴、网格图、立体展开图、概率示意图、集合维恩图、作图题辅助图等。

核心目标：图形只能承载被题目允许承载的数学关系。凡是会改变几何关系、
数量关系、函数性质、比例认知的失真，都属于不可接受错误。

## 2. 真实世界一致性硬约束

### 2.1 禁止项

- 禁止将“不垂直”画成近似垂直、将“不平行”画成近似平行且未声明不按比例。
- 禁止线段端点未闭合、交点错位、角标落在错误角域。
- 禁止坐标轴方向、原点、刻度、单位长度互相矛盾。
- 禁止统计图中数值高度与数据不一致。
- 禁止函数图像与给定解析式、单调性、奇偶性、定义域冲突。
- 禁止立体图的可见边、不可见边表示混乱。
- 禁止圆、弧、切线、割线关系画错但仍作为推理依据。
- 禁止用视觉暗示泄题，例如把正确选项对应图形特别规整或特别突出。
- 禁止非等比例示意图伪装成精确作图。

### 2.2 必须项

- 必须保证所有几何对象可追溯到明确图元定义。
- 必须在非按比例图形上标注“示意图”或等价声明。
- 必须保证点、线、面、角、弧、阴影区域边界清晰且唯一。
- 必须保证同类标记含义一致，例如相等边记号、平行记号、垂直记号统一。
- 必须保证坐标轴刻度单调、单位一致、标注完整。
- 必须保证图中标签与题干命名一一对应，不能重名。

## 3. 几何 / 拓扑 / 语义一致性规则

### 3.1 几何一致性

- 两点确定一线，线段端点必须落在对应点中心容差范围内。
- 标为直角的两线夹角必须在容差内接近 90°。
- 标为平行的直线方向向量夹角必须在容差内接近 0° 或 180°。
- 标为等长的线段若图形声明按比例，长度必须在容差内一致。
- 圆上点必须落在圆周容差带内，圆心连线、半径、直径关系不得冲突。
- 数轴、坐标系上相邻刻度间距必须与数值步长一致。

### 3.2 拓扑一致性

- 几何图必须是合法平面图：交点、端点、重合段拓扑关系明确。
- 封闭图形必须形成闭环，阴影区域必须由闭环边界定义。
- 展开图折叠后应能形成唯一立体，不允许面片重叠冲突。
- 维恩图中交并补区域应与逻辑表达式对应。
- 函数图像不得出现无定义区间上的连续连线。

### 3.3 语义一致性

- 图中标注的数学对象类型必须与题干一致，例如点不能标成线段名。
- 统计图的标题、图例、轴标签、数据系列必须可相互解释。
- 概率示意图中的样本空间和事件区域必须具备包含关系一致性。
- 作图题辅助图只能提供题目允许的已知关系，不得额外补充构造痕迹。

## 4. 自动校验规则（可机器检查）

### 4.1 基础几何校验

- `MA-GR-001`：所有点标签唯一，且在图中存在唯一锚点。
- `MA-GR-002`：线段端点必须绑定到已定义点。
- `MA-GR-003`：闭合多边形首尾点必须重合于容差内。
- `MA-GR-004`：标注直角处夹角需在 90°±2° 范围内。
- `MA-GR-005`：标注平行的线方向差需在 0°±2° 或 180°±2°。
- `MA-GR-006`：坐标轴箭头、原点、刻度、单位标签缺一不可。

### 4.2 图像语义校验

- `MA-GR-101`：函数图像采样点与声明函数在容差内一致。
- `MA-GR-102`：统计图中柱高或折线点位与源数据逐项一致。
- `MA-GR-103`：圆上点到圆心距离偏差不得超过半径的 1%。
- `MA-GR-104`：阴影区域必须由合法闭合边界包围，不得泄漏到外部。
- `MA-GR-105`：所有相等记号组必须指向同类对象。

### 4.3 版面与可读性校验

- `MA-GR-201`：任何标签不得与边、点、角弧发生遮挡。
- `MA-GR-202`：密集图中最小标签间距不得小于预设阈值。
- `MA-GR-203`：导出后 OCR 识别点名、轴名、单位名必须与源数据完全一致。

## 5. 渲染前校验、渲染后回归检查、失败处理策略

### 5.1 渲染前校验

- 从结构化几何数据出发，不接受仅凭位图拼接出题。
- 先做几何求值与约束求解，再做绘制。
- 校验所有对象命名、约束、关系、可见性与题干一致。
- 校验“是否按比例”标记与当前绘制模式一致。
- 校验标签避让结果，避免标注压线。

### 5.2 渲染后回归检查

- 对几何图执行拓扑重建，复算交点、角度、边界闭合性。
- 对函数图、统计图执行像素采样和数据反推比对。
- 对关键模板执行截图回归，防止样式更新破坏标记位置。
- 检查模糊、锯齿、裁切、低对比度与缺线问题。

### 5.3 失败处理策略

- 约束不可满足时，直接回退到重新求解布局或重新生成图形。
- 不允许把精确图强行降级成“差不多”的手绘风示意图。
- 不允许删掉关键标签、移除刻度、去除相等标记作为规避方案。
- 多次失败后，换用同知识点的其他题模板，不输出错误图。

## 6. 学段差异

### 6.1 小学

- 以数轴、基础平面图形、简单统计图、应用题情境图为主。
- 必须减少标签密度，不使用复杂立体透视和多重约束图。
- 允许更强的视觉辅助，但不得引入答案提示。

### 6.2 初中

- 覆盖三角形、四边形、圆、一次函数、反比例函数、统计与概率。
- 必须强化角、边、坐标刻度的准确性，禁止“看起来像”式作图。
- 几何证明图要明确已知与求证，不多给隐含关系。

### 6.3 高中

- 覆盖解析几何、导数图像、立体几何、数列、概率统计、向量。
- 必须支持多图层、复杂标注和高精度坐标关系。
- 非按比例图更容易误导，必须显式标记并加强文字约束说明。
